**leetcode 141**

如果链表有环，那么快指针一定会与慢指针相遇。

```java
public class Solution {
    public boolean hasCycle(ListNode head) {
        if(head == null || head.next == null){
            return false;
        }

        ListNode slow = head;
        ListNode fast = head.next;

        while(slow != fast){
            slow = slow.next;
            if(fast == null || fast.next == null){
                return false;
            }
            fast = fast.next.next;
        }

        return true;
    }
}
```

核心代码也可以改为如下

```java
while(fast != null && fast.next != null){
    if(slow == fast){
        return true;
    }
    slow = slow.next;
    fast = fast..next.next;
}
```
