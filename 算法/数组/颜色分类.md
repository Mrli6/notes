# leetcode 75



## 解法一

```java
class Solution {
    public void sortColors(int[] nums) {
        int zero = 0;   //记录0的个数
        int one = 0;    //记录1的个数
        int two = 0;    //记录2的个数

        for(int num : nums){
            if(num == 0){
                zero++;
            }else if(num == 1){
                one++;
            }else{
                two++;
            }
        }

        //填充0、1、2
        if(zero != 0){
            Arrays.fill(nums, 0, zero, 0);
        }
        if(one != 0){
            Arrays.fill(nums, zero, zero+one, 1);
        }
        if(two != 0){
            Arrays.fill(nums, zero+one, zero+one+two, 2);
        }
    }
}
```
